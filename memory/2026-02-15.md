## 浏览器发帖成功经验（X/Twitter）

### 成功关键步骤
1. 先启动 relay 服务：`browser start --profile chrome`
2. 用户手动点击 Chrome 扩展图标（确保 badge 显示 ON）
3. 使用 `profile="chrome"` 而非 `profile="openclaw"`
4. 直接操作已登录的 Chrome 标签页

### 失败教训
- 用 `profile="openclaw"` 会开新浏览器，没有登录状态
- 扩展未 attach 时无法连接
- 每次操作后标签页 ID 会变，需要重新获取
